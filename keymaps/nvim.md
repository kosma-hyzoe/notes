## General

* `[s/]s` - next/previous spell error
* `,nc` - "no comment"
* `J/K` - move selection up/down

## LSP

* `K` - info
* `,wvs` - workspace_symbol
* `,od` - open diagnostic (`[d/]d` - go to next/prev)
* `,vca` - code actions
* `,vrr` - references
* `,vrn` - rename
* `<c-h>` - signature help

## Telescope

* `,lg` - live grep
* `,gf` - git files
* `,ff` - find files
* `,tg` - tags
* `,gg` - grep string
* `,ht` - help tags
* `,mp` - man pages
* `,tss` - spell suggest
